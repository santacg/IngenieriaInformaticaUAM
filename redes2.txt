  2. Las direcciones IP asignadas a cada interfaz, 
  ¿son públicas o privadas? ¿Cómo puedes identificar cada una?
  Para identificar si una dirección IP es pública o privada, 
  se pueden utilizar los rangos definidos para direcciones IP privadas en IPv4, que son:

  10.0.0.0 a 10.255.255.255
  172.16.0.0 a 172.31.255.255
  192.168.0.0 a 192.168.255.255

  Ambas interfaces tienen sus direcciones ip fuera de estos rangos con lo cual son 
  privadas
